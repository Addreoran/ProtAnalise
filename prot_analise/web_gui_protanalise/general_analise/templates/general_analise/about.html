Idea of Protein Analiser

Protein Analiser was created to tidy all my scripts to analise and allow other people to use them. It will be contributed with my science work.

Actual features

Gui of Protein Analiser allow to load data in fasta format separated to clusters with certain headers. Headers of clusters can be specify in loading data Actual features allows group data in:
  1. Species
  2. Kingdoms
  3. Proteins

Species

Proteins in clusters can be separated into species. Species names comes from selected in loading dataset to use database.

Kingdoms

Proteins in clusters can be separated into kingdoms. Kingdoms names comes from selected in loading dataset to use database.

Proteins

Regions in clusters can be also separated into proteins. Protein id comes from headers of loaded sequences.

Future features

In the futher work, there will be also analysed all sequences. There will be also added analysis of conserved aminoacids.
